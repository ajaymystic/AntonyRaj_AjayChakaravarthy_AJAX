# Novo EarBuds - Interactive AR Promotional Page

A dynamic, responsive promotional website featuring an interactive 3D AR model of Novo EarBuds with real-time content loading via AJAX and the Fetch API.

![Novo EarBuds Preview](images/poster.webp)

## üéØ Project Overview

This project is an interactive promotional page for Novo EarBuds, built as part of the Web Development 2 (MMED-3040) course at Fanshawe College. The page features a 3D AR model viewer with interactive hotspots, dynamic content loading from external APIs, and a fully responsive design that adapts seamlessly from mobile to desktop.



## üõ† Technologies Used

### Frontend
- **HTML5**: Semantic markup and template elements
- **CSS3/SASS**: Modern styling with modular architecture

- **GSAP 3.6.1**: GreenSock Animation Platform for smooth animations

- **Earbuds Info API**: `https://swiftpixel.com/earbud/api/infoboxes`
- **Materials API**: `https://swiftpixel.com/earbud/api/materials`

- **Git/GitHub**: Version control and collaboration
- **SASS Compiler**: CSS preprocessing
- **VS Code**: Primary development environment
- **Live Server**: Local development server
